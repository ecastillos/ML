# Roadmap inicial

## Fase 1: MVP navegable

- Home con búsqueda visual
- Listado de productos
- Detalle de producto
- Carrito local
- API de productos mock
- Modelo inicial de base de datos

## Fase 2: Usuarios y vendedores

- Registro e inicio de sesión
- Roles: comprador, vendedor y administrador
- Panel para publicar productos
- Subida de imágenes
- Inventario básico

## Fase 3: Compra y órdenes

- Carrito persistente
- Creación de órdenes
- Estados de orden
- Historial de compras
- Notificaciones básicas

## Fase 4: Marketplace avanzado

- Filtros por categoría, precio y ubicación
- Reputación de vendedores
- Preguntas y respuestas por producto
- Favoritos
- Reviews

## Fase 5: Producción

- Seguridad y validaciones completas
- CI/CD
- Observabilidad
- Backups
- Pasarela de pagos
- Política legal, privacidad y términos
